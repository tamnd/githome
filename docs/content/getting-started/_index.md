Start with the [quick-start guide](/getting-started/quick-start/) to run githome
locally, then [connect your tools](/getting-started/connect-your-tools/) so gh
CLI, VS Code, Octokit, and Terraform all point at your instance.

When you are ready to run githome in production, see the [Deploy](/deploy/)
section for Docker Compose, HTTPS, and backup guides.

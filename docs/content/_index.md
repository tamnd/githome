---
title: "githome"
description: "A self-hosted GitHub-compatible forge: one binary, full API compatibility."
---

githome is a self-hosted git forge that speaks the GitHub REST API v3 and
GraphQL v4 natively. Point any GitHub client at it and it works: gh CLI,
Octokit, Terraform, VS Code, JetBrains, git over HTTP.

One binary. No external services required. SQLite by default, PostgreSQL when
you need it.
